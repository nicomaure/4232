Crea Tus Propios Mundos

CoSpaces Edu

Crea mundos 3D, animaciones y juegos. Ideal para contar historias o hacer simulaciones. ¡Puedes programar con bloques!

Plan Gratuito: Genial para empezar. Permite 1 clase con 29 alumnos, pero cada uno puede guardar solo 2 proyectos. ¡Perfecto para trabajos finales!
Empezar a Crear

Assemblr EDU

Arma escenas 3D fácilmente arrastrando y soltando objetos. Muy bueno para hacer presentaciones y maquetas virtuales.

Plan Gratuito: Bueno para proyectos chicos o para probar. Tiene poco espacio para guardar (30MB), ¡así que úsalo con inteligencia!
